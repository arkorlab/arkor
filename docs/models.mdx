Today Studio's Playground (base-model mode) and Arkor's hosted inference serve exactly one base model: `gemma-4-E4B-it`. The `model` field of `createTrainer` itself accepts any model identifier the training backend can load, but the starter templates target Gemma 4 and only registered models can be chatted with after training. Below: what that value gets you and what's next.

## What you can pick today

`gemma-4-E4B-it` is the Gemma 4 instruction-tuned build, packaged by Unsloth for fast LoRA / QLoRA fine-tuning. Every starter template (`triage`, `translate`, `redaction`) targets it by default.

```ts
import { createTrainer } from "arkor";

export const trainer = createTrainer({
  name: "support-bot-v1",
  model: "unsloth/gemma-4-E4B-it",
  dataset: { type: "huggingface", name: "arkorlab/triage-demo" },
});
```
